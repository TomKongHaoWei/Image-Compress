# 🖼️ 在线图片压缩工具 (Image Compress)

## ✨ 项目简介
这是一个简洁高效的**纯前端**图片体积优化（压缩）工具。它完全基于 **JavaScript** 和浏览器原生 `<canvas>` API 实现，用户上传图片后无需经过服务器处理，即可在浏览器本地完成压缩并下载。

此工具特别适用于用户在上传大尺寸照片（如手机原图）前进行快速优化，以节省带宽和存储空间。

## 💡 主要功能

* **📥 导入图片：** 用户可以通过按钮选择本地图片文件。
* **⚙️ 自动压缩：** 图片导入成功后，立即在浏览器端自动启动压缩流程。
    * **尺寸限制：** 将图片最大宽度限制在 `1200px`（可配置）。
    * **质量调整：** 强制导出为 JPEG 格式，并将质量设置为 `0.7`（可配置）。
* **💾 下载优化文件：** 压缩完成后，自动启用下载按钮，用户可以立即获取优化后的 JPEG 格式图片。
* **🔒 隐私保护：** 所有处理均在客户端完成，图片数据不会上传至任何服务器。

## 🛠️ 技术栈

* **HTML5 / CSS3：** 负责页面结构和基础样式。
* **原生 JavaScript：** 实现核心逻辑，包括文件读取 (`FileReader`)、图片加载 (`Image`) 和图像处理 (`Canvas API`)。

## 🚀 如何使用

1.  将 `index.html` 和 `compressor.js` 放在同一个文件夹内。
2.  在任何现代浏览器中打开 `index.html` 文件。

### 操作步骤：

1.  点击 **📥 导入图片并压缩** 按钮，选择您想要优化的图片文件（支持 `.jpeg`, `.png`）。
2.  程序将自动在后台进行压缩，并在状态栏显示原始大小和压缩后的大小对比。
3.  压缩完成后，点击 **💾 下载压缩后的图片** 按钮即可保存优化后的 JPEG 文件。

## ⚙️ 核心配置（`compressor.js`）

您可以根据需求修改 `compressor.js` 文件顶部的两个核心参数，以调整压缩效果：

| 变量名 | 描述 | 推荐范围 |
| :--- | :--- | :--- |
| `MAX_WIDTH` | 图片允许的最大宽度（像素）。这是最有效的瘦身方法。 | 800 - 1920 |
| `JPEG_QUALITY` | JPEG 图片导出的压缩质量。值越低，体积越小，画质越差。 | 0.6 - 0.8 |
